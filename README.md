# CrazyCatLadies

## HTML/CSS
Create a single webpage from an HTML file that can be viewed locally from a browser

The HTML file should include both "Internal CSS" (\<style\>) and "Inline CSS" (<\p style="color: red; font-size: 20px;"\>)

The webpage should include:

- large hero text at the top of the page that spans the width of the page
- 2 cat images taken from online
- an ordered list (numbered) of 5 different cat facts
- Create a version 2 of the webpage by moving all CSS into an "External" css file
